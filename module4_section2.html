GPT로 매일 아침, 건설 뉴스 브리핑 받기
간단한 명령으로 정보 수집 시간을 확 줄여보세요!
01

ChatGPT와 같은 대화형 AI에게 주기적인 작업을 설정해두면, 매일 아침 자동으로 원하는 정보를 요약하여 받아볼 수 있습니다.
예를 들어, 건설 관련 주요 뉴스 사이트의 최신 기사를 확인하고 정리해달라고 요청할 수 있습니다.

✨ 이런 점이 좋아요!

  • 📰 매일 아침 최신 건설 뉴스를 놓치지 않고 확인 가능
  • ⏱️ 여러 사이트를 방문할 필요 없이 한 곳에서 요약 정보 취득
  • 🎯 중요한 키워드와 함께 핵심 내용만 빠르게 파악
  • 🔗 원문 링크 제공으로 필요시 상세 내용 즉시 확인

활용 Tip: ChatGPT의 'Scheduled Send' 기능(유료 플랜), 유사 확장 기능, 또는 AI 모델 API와 스크립트를 연동하여 자동화할 수 있습니다. (세부 방법은 사용 중인 AI 모델 및 서비스 플랜에 따라 다를 수 있습니다.)

자동화 툴 연동: 국토부 공지, 슬랙으로 실시간 알림!
Make, Zapier 등 노코드 툴로 중요 정보를 놓치지 않고 받아보세요.
02

Make(구 Integromat), Zapier, n8n과 같은 워크플로우 자동화 툴을 사용하면,
국토교통부 웹사이트의 새로운 공지사항을 주기적으로 확인하고, AI로 요약하여 팀 메신저(예: Slack)로 자동 발송할 수 있습니다.

🚀 자동화의 강력한 효과!

  • 🔔 중요한 정부 공지사항 및 보도자료를 실시간으로 파악
  • 🤖 AI가 핵심 내용을 요약해주어 정보 확인 시간 단축
  • 💬 팀원 전체에게 슬랙 등 메신저로 즉시 공유하여 빠른 대응 가능
  • 🔄 반복적인 수동 확인 업무를 제거하여 업무 효율성 극대화

활용 Tip: 자동화 툴에서 웹사이트 RSS 피드를 모니터링하거나, HTTP 요청으로 웹페이지 내용을 주기적으로 가져와 변경 사항을 감지합니다.
새로운 내용이 감지되면 OpenAI API 등을 호출하여 요약 후, 슬랙 메시지로 전송하는 워크플로우를 구성합니다.
